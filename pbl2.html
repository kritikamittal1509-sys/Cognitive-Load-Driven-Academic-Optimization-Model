Cognitive Load–Driven Academic Optimization Model

This project investigates how study behavior patterns and cognitive load indicators influence academic performance and proposes a machine learning-based framework to generate intelligent, personalized learning recommendations.

Problem Statement

Most academic systems focus solely on predicting marks using attendance and study hours. However, students with similar study durations often achieve different outcomes. This indicates that performance is influenced not only by study quantity but also by cognitive effort and learning efficiency. There is limited research integrating study pattern analytics with cognitive load estimation into a unified analytical framework.

Literature Review / Market Research

Recent studies in Learning Analytics and Educational Data Mining analyze academic performance using behavioral metrics. Separate research explores cognitive load estimation through interaction features such as click frequency and pause duration using ML models.

Research Gap / Innovation

Existing systems treat performance prediction and cognitive load estimation independently. Our project integrates both domains to provide deeper insight into why performance varies and how study efficiency can be optimized.

System Methodology

Dataset

Student data including study hours, consistency index, attendance, break frequency, session duration, click rates, pause durations, and task switching frequency. Data preprocessing includes normalization and feature scaling.

ML Models

Random Forest and Support Vector Machine are used for cognitive load classification, while Linear Regression is applied for academic performance estimation. Correlation analysis identifies significant behavioral predictors.
